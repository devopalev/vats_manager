# vats_manager

<h2>Requirements</h2>

- Python 3.8+
- openpyxl
- xlsxwriter
- pandas
- python-telegram-bot
- google-api-python-client 
- google-auth-httplib2 
- google-auth-oauthlib
- oauth2client
- bs4
- dokuwiki
- lxml
- websockets

Use command *pip3 install --upgrade* to install Python package

If pip3 are not installed: *sudo apt-install python3-pip*
